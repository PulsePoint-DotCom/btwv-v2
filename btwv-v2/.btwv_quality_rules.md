
## Language Rule
- **All content must be in English only** — no Chinese characters, no non-ASCII text
- Before deploying new articles, run this check:
  ```python
  import re, os
  chinese = re.compile(r'[\u4e00-\u9fff]')
  for f in os.listdir('.'):
      if f.endswith('.html'):
          with open(f) as fh:
              if chinese.search(fh.read()):
                  print(f'NON-ENGLISH: {f}')
  ```
